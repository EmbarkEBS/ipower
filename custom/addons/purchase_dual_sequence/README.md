# Purchase RFQ & PO Separate Sequences

## Features

- Separate RFQ numbering
- Separate Purchase Order numbering
- RFQ numbers remain independent from PO numbers

## Example

RFQ0001
RFQ0002

Direct PO:
PO0001
PO0002

Confirm RFQ0001:
PO0003

## Installation

1. Copy module to custom addons path
2. Restart Odoo
3. Update Apps List
4. Install module

## Compatible Version

- Odoo 19
